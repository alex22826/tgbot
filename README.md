# tgbot — Telegram-мост к Claude Code

Личный Telegram-бот, который передаёт ваши сообщения локальному Claude Code CLI.
Умеет всё, что Claude Code в терминале: создавать репозитории и сайты, пушить на
GitHub, править файлы, искать в интернете (погода, новости и т.д.).

## Установка

1. В Telegram напишите [@BotFather](https://t.me/BotFather): `/newbot`, придумайте
   имя — получите токен вида `123456:ABC-DEF...`.
2. Откройте файл `.env` и вставьте токен:
   `TELEGRAM_BOT_TOKEN=ваш_токен`
3. Запустите `start.bat` (нужны Python 3 и авторизованный Claude Code).
4. Напишите боту в Telegram `/start` — **первый написавший становится владельцем**,
   все остальные пользователи игнорируются.

## Команды

- `/new` — начать разговор заново (сбросить контекст)
- `/help` — справка

## Важно про безопасность

- Бот запускает Claude Code **без подтверждений** (`--dangerously-skip-permissions`) —
  он реально выполняет команды на компьютере. Токен из `.env` никому не показывайте:
  кто владеет токеном до первого `/start`, тот может стать владельцем бота.
- Рабочая папка Claude — `E:\codeAI\botspace`, проекты создаются там.
- Бот работает, пока включён компьютер и запущен `start.bat`.
